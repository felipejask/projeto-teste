Para executar o projeto deve-se:

1) Abrir atravez do terminal o arquivo do projeto;
2) Instalar as dependências:
	-npm/Node.js
	-Angular CLI
	-Bootstrap
	-Ng Bootstrap
	-JQuery
3) Rodar o comando ng serve -o para abrir uma janela com a aplicação ou somente ng serve caso uma janela da aplicação já esteja aberta mas a aplicação não esteja rodando.
